Pre-Requisite
  1. Install Node & Configure.
  2. Install Visual Studio Code.

  Open Visual Studio Code.
  1. Install Playwright. Use Command: npm init playwright.
  2. Keep Compiler - JavaScript & Test Folder = true.
  3. Pull Git Repository: Hotel.

Execution:
  1. Open Terminal > Navigate to the Project folder Path.
  2. Execute command: npx playwright test HotelTest.spec.js
